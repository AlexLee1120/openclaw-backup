# SRS Pro 銷售漏斗與 Landing Page 企劃 (2026-03-25)

## 🎯 漏斗架構 (Marketing Funnel)
1. **流量來源 (Top of Funnel):** YouTube Shorts (如 EP1), IG Reels。
2. **登陸頁 (Landing Page):** 負責痛點共鳴、展示產品價值，並提供免費誘餌 (Lead Magnet)。
3. **名單擷取 (Lead Capture):** 使用者輸入 Email 領取「30 字動漫單字先鋒版」。
4. **免費體驗 (Activation):** 系統自動寄送帶有專屬 Token 的體驗連結。
5. **付費轉化 (Monetization):** 體驗結束後 (或信件跟進)，導流至「解鎖 1000 字完整圖鑑版」的購買頁面 (串接 ECPay)。

## 📄 Landing Page 結構設計 (一頁式長網頁)

### Section 1: 破題與痛點 (Hero & Pain Points)
- **大標題:** 「昨天背的 7000 單，今天又全忘了嗎？」
- **副標題:** 「丟掉厚重的參考書！這是一款專為台灣學測設計，結合『動漫圖鑑』與『SM2 科學記憶法』的提分外掛。」
- **痛點共鳴 (3大痛點):**
  1. 🎒 書包太重：通勤想背單字，卻不想拿書出來。
  2. 📉 缺乏效率：沒有複習計畫，永遠在背 A 開頭的 abandon。
  3. 🥱 過程枯燥：黑白文字太無聊，背到睡著。

### Section 2: 解決方案與產品優勢 (Solution & Benefits)
- **大字:** 「你的手機，就是最強的單字神器。」
- **三大核心功能 (Feature & Benefit):**
  1. **🧬 矽谷級 SM2 演算法:** 系統比你還了解你的大腦，自動在「快要忘記的那一刻」推播複習。效率提升 300%。
  2. **🎨 AI 動漫視覺圖鑑:** 每個單字都有專屬的高品質動漫插畫，用視覺聯想秒殺死記硬背。
  3. **👆 抖音式滑動操作 (Swipe):** 左滑忘記、右滑記得。單手就能刷單字，捷運上也能輕鬆複習 100 題。

### Section 3: 社會認同與權威 (Social Proof)
- **權威背書:** 「科技公司 CTO 親自研發的數位化學習法，讓大腦像電腦一樣高效運作。」
- **預期成效:** 每天 15 分鐘，30 天搞定學測核心高頻單字。

### Section 4: 免費誘餌與行動呼籲 (Lead Magnet & CTA)
- **標題:** 「限量開放！免費解鎖【學測必考核心 30 單字體驗版】」
- **表單:** [輸入您的 Email]
- **按鈕:** [立即領取我的提分外掛 🚀]
- **備註:** 「體驗版包含完整 30 張動漫卡牌與 SM2 記憶追蹤功能。完全免費。」

### Section 5: 付費升級預告 (Upsell Teaser - 放在領取成功頁面)
- 當用戶輸入 Email 後，跳轉到 `success.html`：
- 「恭喜！體驗連結已寄至您的信箱。」
- 「覺得 30 字不夠過癮？【SRS Pro 完整 1000 字動漫圖鑑版】現正早鳥預購中，結帳輸入折扣碼即享優惠... [了解更多]」

## 🛠️ 執行技術方案
1. **前端頁面:** 使用 Tailwind CSS 快速刻出這個一頁式 HTML (`index.html`)。
2. **表單提交:** 透過 AJAX (Fetch API) 將 Email 送到我們在 Railway 上架設的 Python Flask 後端。
3. **後端處理:** Railway 收到 Email，寫入 SQLite 資料庫（收集名單），並呼叫 Resend API 寄出體驗連結。
4. **付費金流:** 串接 ECPay (綠界)，購買後解鎖 1000 字的 CSV 權限。
