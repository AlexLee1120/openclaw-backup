# YouTube Shorts 生成流程 (2026-03-25)

## AI (小艾與智能體團隊) 可以「全自動」負責的部分：
1. **主題企劃：** 發想痛點、爆款標題。
2. **腳本撰寫 (Script)：** 精準控制在 60 秒內，分為破題 (Hook)、內容 (Body)、行動呼籲 (CTA)。
3. **分鏡設計 (Storyboard)：** 建議每一個畫面的視覺元素與字卡 (B-roll & Text overlays)。
4. **配音音檔 (TTS)：** 使用 `elevenlabs` 或 `microsoft` 語音套件直接為您產出專業的旁白音檔 (mp3/wav)。
5. **視覺素材 (AI 繪圖)：** 使用 Gemini Image Gen (1K/2K 解析度) 產生可用的背景圖或插畫素材。

## 需要 Alex (或助理)「手動」執行的部分：
1. **剪輯軟體組裝：** 將我產出的「旁白音檔」與「圖片素材」，放入剪映 (CapCut) 或 Premiere 之中對齊時間軸。
2. **上字幕：** 在剪映中使用「自動辨識字幕」功能，套用動態效果。
3. **手機實錄 (部分需要)：** 如果腳本有要求「實測操作 SRS Pro 畫面」，這部分會需要您花 30 秒用手機螢幕錄影。
4. **上傳 YouTube：** 因為涉及 2FA 安全認證，發布的最後一步需由人類點擊。
