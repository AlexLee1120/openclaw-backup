# 霹靂布袋戲「大轉骨」戰略重塑報告摘要 (2026-03-23)

## 1. 轉型願景
霹靂正推動「大轉骨」計畫，目標是從傳統家族戲班轉型為高度專業化的「泛娛樂」IP 帝國，並預計在 2050 年達成碳中和與文化永續。

## 2. 核心支柱
- **組織現代化：** 拆分內容與管理中心，強化智權（IP）管理與法務確權。
- **品牌再造：** 推出 "Keep on rolling" 標語，視覺設計扁平化，將角色特徵符號化（圖騰系統）。
- **技術革命：** 導入 AI 語音模擬、3D 列印戲偶、4K/8K 規格，並進軍 Web3/NFT 市場。

## 3. 全球化與敘事
- **東離模式：** 透過「製作委員會」與日本深度捆綁，降低文化進入門檻。
- **宇宙重啟：** 電影《素還真》將英雄人性化，並實施正劇與重啟版（英雄戰紀）的雙軌策略。

## 4. 財務與挑戰
轉型期面臨財務虧損與新舊戲迷價值觀衝突（傳統韻味 vs. 現代化改革）。
